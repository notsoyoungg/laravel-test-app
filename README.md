## Разворачивание приложения

 Проект можно развернуть одной командой:<br>
```bash
    make setup
```

Или же последовательно выполните следующие команды:

1. Установите зависимости PHP через Composer:

   ```bash
   composer install
2. Скопируйте пример конфигурационного файла .env:
    ```bash
   cp .env.example .env
3. Сгенерируйте ключ приложения Laravel:
   ```bash
   php artisan key:generate
4. Соберите Docker-контейнеры:
    ```bash
   docker compose build
5. Запустите приложение в фоне:
    ```bash
   docker compose up -d

## Форматирование кода

Для локальной разработки установлен Laravel Pint<br>
Для форматирования кода в соответствии с PSR-12 выполните команду:
```bash
sh pint.sh
```
